FIORENZA PIZZARIA E RESTAURANTE================================Sistema em Flask com:- Login por telefone + senha 6 dígitos- Carrinho- Checkout Stripe- Webhook Stripe- Admin completo- Upload de imagens- SQLite local- Preparado para PostgreSQL================================1) INSTALAÇÃO LOCAL================================1. Instalar Python 3.10+2. Criar ambiente virtual:python -m venv venv3. Ativar ambiente:Windows:venv\Scripts\activateMac/Linux:source venv/bin/activate4. Instalar dependências:pip install flask flask_sqlalchemy stripe gunicorn python-dotenvOu se existir:pip install -r requirements.txt================================2) CONFIGURAR VARIÁVEIS (STRIPE)================================Criar arquivo .env na raiz do projeto:STRIPE_SECRET_KEY=sk_test_...STRIPE_PUBLISHABLE_KEY=pk_test_...STRIPE_WEBHOOK_SECRET=whsec_...BASE_URL=http://127.0.0.1:5000================================3) CRIAR BANCO DE DADOS================================flask init-dbOU apenas rodar:python app.py(O sistema cria automaticamente)================================4) RODAR LOCAL================================python app.pyAcessar no navegador:http://127.0.0.1:5000Para acessar pelo celular na mesma rede:1) Descobrir IP:ipconfig2) Rodar app já configurado com:app.run(host="0.0.0.0", port=5000, debug=True)3) No celular:http://SEU_IP:5000================================5) TESTAR STRIPE LOCAL================================Instalar Stripe CLI:https://stripe.com/docs/stripe-cliLogin:stripe loginRodar servidor:python app.pyEm outro terminal:stripe listen --forward-to localhost:5000/webhook/stripeCopiar o "whsec_..." gerado e colocar no .envCartão de teste:4242 4242 4242 4242Validade: 12/30CVC: 123Para simular pagamento:stripe trigger checkout.session.completed================================6) RODAR EM PRODUÇÃO (SERVIDOR LINUX)================================Instalar:sudo apt updatesudo apt install python3 python3-pip python3-venv nginxCriar venv:python3 -m venv venvsource venv/bin/activateInstalar dependências:pip install -r requirements.txtRodar com gunicorn:gunicorn -w 4 -b 0.0.0.0:8000 app:appConfigurar NGINX:sudo nano /etc/nginx/sites-available/fiorenzaConteúdo:server {    listen 80;    server_name seudominio.com;    location / {        proxy_pass http://127.0.0.1:8000;        proxy_set_header Host $host;        proxy_set_header X-Real-IP $remote_addr;    }}Ativar:sudo ln -s /etc/nginx/sites-available/fiorenza /etc/nginx/sites-enabledsudo systemctl restart nginx================================7) HTTPS (RECOMENDADO)================================sudo apt install certbot python3-certbot-nginxsudo certbot --nginx -d seudominio.com================================8) ESTRUTURA IMPORTANTE================================Certifique-se que exista:static/uploads/static/images/Permissão no servidor:sudo chmod -R 755 static/uploads================================9) STATUS DO STRIPE================================Pedido começa como:aguardando_pagamentoApós webhook confirmar:pagoWebhook:POST /webhook/stripe================================10) PRONTO================================Sistema online e funcionando.Admin:http://seudominio.com/adminSenha
